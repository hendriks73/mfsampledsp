MFSampledSP

Media Foundation based service provider for javax.sound.sampled.

Download this project as a .zip file Download this project as a tar.gz file

Welcome!

MFSampledSP is a free implementation of the javax.sound.sampled service provider interfaces based on Microsoft's Media Foundation library.

Its main purpose is to decode audio files or streams to linear PCM.
